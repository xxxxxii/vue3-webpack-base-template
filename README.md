# vue3-webpack-base-template

vue3 + webpack 基础项目模板

## 已配置项目

1. vue-router beforeEach，afterEach
2. axios
3. eslint + prettier
4. env 环境变量
5. NProgress

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
